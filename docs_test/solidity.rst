Solidity智能合约
================

1.概述
------

Hyperchain区块链平台支持多级密码学加密来保证数据安全，利用了以下密码学算法来保证了数据的安全性问题：

-  椭圆曲线数字签名:secp256k1,secp256r1
-  对称加密算法:3DES,AES
-  密钥交换:ECDH
-  密码杂凑算法:Keccak-256
